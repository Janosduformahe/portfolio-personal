# Portfolio Personal — Janos Buzas

Portfolio web de una sola página para Janos Buzas, Fintech & AI Developer.

## Stack
- HTML5 semántico + SEO
- CSS vanilla con design system (glassmorphism, dark premium)
- JavaScript vanilla (sin dependencias)

## Secciones
1. **Hero** — nombre, typewriter animado, canvas de partículas, stats KPI
2. **Sobre mí** — bio + stack tecnológico
3. **Proyectos** — 4 proyectos con filtros por categoría (AI / Fintech / AgriTech)
4. **Experiencia** — timeline vertical animada
5. **Contacto** — formulario validado + links sociales

## Proyectos incluidos
- [IBZ AI Landing](https://ibz-ai-landing.vercel.app/) — Proyecto propio
- [Purchase Order System](https://purchaseorder-one.vercel.app/login) — Fintech
- [Juntos Sales Dashboard](https://juntossales.vercel.app/) — Fintech
- [AgroFlow ERP](https://agroflow-p44i.vercel.app/) — AgriTech + IA

## Deploy
Arrastra esta carpeta a [vercel.com/new](https://vercel.com/new) — funciona sin configuración.

> **Nota**: Añade tu foto como `photo.jpg` en la raíz del proyecto.
